# Gerenciador de Produtos (CRUD)

Projeto desenvolvido para treinar e solidificar conhecimentos em JavaScript, Manipulação de DOM e estruturação de arquitetura de software.

##  Objetivo
O principal foco deste projeto foi aplicar na prática o padrão de arquitetura MVC (Model-View-Controller) utilizando apenas JavaScript puro, sem frameworks. O objetivo foi entender a fundo como os dados trafegam entre a lógica de negócio e a interface do usuário.

## Tecnologias e Conceitos Aplicados
- **HTML5 & CSS3**: Uso de HTML e Flexbox para layout.
- **JavaScript**:
  - POO (Programação Orientada a Objetos) com Classes.
  - Manipulação avançada do DOM.
  - Padrão MVC: Separação clara de responsabilidades (Model, View, Controller).
  - Event Delegation (Delegação de eventos para elementos dinâmicos).
  - Métodos de Array.

## Funcionalidades
- [x] **Adicionar**: Criação de novos produtos com validação de campos.
- [x] **Listar**: Renderização dinâmica da tabela de produtos.
- [x] **Remover**: Exclusão de itens da lista/memória.
- [ ] **Editar**: Atualização de dados existentes (Em desenvolvimento).
- [ ] **Polimento**: Mais treinos profissionalizando o projeto.

## Como rodar o projeto
1. Clone este repositório.
